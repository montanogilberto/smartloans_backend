Send a ticket notification by WhatsApp.

Request body:
- phone (required): E.164 format (example: +521234567890)
- message (optional): custom message body
- receiptUrl (optional): URL appended to the message

This endpoint sends outbound WhatsApp messages through Twilio WhatsApp API.
Requires environment variables:
- TWILIO_ACCOUNT_SID
- TWILIO_AUTH_TOKEN
- TWILIO_WHATSAPP_FROM (example: whatsapp:+14155238886)
