Returns all suppliers records for the given company filtered by companyId via sp_suppliers_all.
Stored procedure: EXEC [dbo].[sp_suppliers_all] @pjsonfile = '<json>'

Example:
curl -X POST "https://smartloansbackend.azurewebsites.net/all_suppliers"   -H "Content-Type: application/json"   -d '{"companyId":1}'
