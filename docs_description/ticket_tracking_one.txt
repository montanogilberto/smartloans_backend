Creates, validates, and updates ticket tracking information using the `sp_ticket_tracking` stored procedure.

Expected payload format:
{
  "ticket": [
    {
      "action": "validate | save | whatsapp | sms",
      "incomeId": 123,
      "companyId": 1,
      "fileName": "receipt_123.html",
      "containerName": "ticketspos",
      "receiptUrl": "https://.../receipt_123.html",
      "phone": "+521000000000"
    }
  ]
}

Actions:
- `validate`: returns ticket tracking row(s) for the provided `incomeId`.
- `save`: inserts/updates receipt metadata and marks Azure upload info.
- `whatsapp`: marks WhatsApp sent status/date/phone.
- `sms`: marks SMS sent status/date/phone.

Returns JSON generated by the stored procedure.
